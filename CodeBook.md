# Code Book
Summary  of the dataset tidy.txt generated by run_analisis.R (script)

## Identifiyng the type of test and Subject tested

*`Subject`:this column shows the id of the subject submitted to the test, id's fit in this range: [1,30]

*`Activity`:this column shows the type of test the subject was submited to, the activities are shown below:

        + WALKING
        + WALKING_UPSTAIRS
        + WALKING_DOWNSTAIRS
        + SITTING
        + STANDING
        + LAYING
